whatever
========

No pull requests will be accepted. Let's see where it goes!

## Idea, whatever...

* pick any language you want
* write anything you want
* create a pull request
* let's see what's possible to build this way!

## How to run, whatever...

Copy this into a terminal:
```sh
$ sudo curl https://raw.githubusercontent.com/tomekw/whatever/master/whatever | bash
Hello, whatever
```
